# Getting Started

To get started with the repository, follow these steps:

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/chirag876/.NETWorkSpace.git
    ```

2. **Navigate to the Project Directory:**
    ```bash
    cd .NETWorkSpace
    ```

3. **Build the Project:**
    ```bash
    dotnet build
    ```

4. **Run the Project:**
    ```bash
    dotnet run
    ```
